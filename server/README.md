### Voraussetzungen

* Python 3.7.x
  * python-socketio
  * gphoto2
* gphoto2
* libgphoto2-dev

Zum Installieren von *gphoto2* und *libgphoto2-dev* muss folgender Befehl in der Kommandozeile eingegeben werden:

```bash
sudo apt-get install gphoto2 libgphoto2-dev
```
